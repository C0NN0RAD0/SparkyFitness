# Proxmox Installation Guide

This guide covers how to install SparkyFitness on Proxmox VE in an LXC container using the included install script.

## Quick Install

Run the following command in the **Proxmox VE shell** to create a new LXC container and install SparkyFitness from this repository:

```bash
bash -c "$(wget -qLO - https://raw.githubusercontent.com/C0NN0RAD0/SparkyFitness/main/install/proxmox.sh)"
```

This script:
- Creates a Debian 13 LXC container (2 CPU cores, 2 GB RAM, 4 GB disk by default)
- Installs Node.js, pnpm, nginx, and PostgreSQL inside the container
- Builds and deploys the SparkyFitness frontend and backend
- Configures and starts all required services automatically

Once complete, SparkyFitness will be accessible at `http://<container-ip>`.

## Updating

To update an existing installation to the latest release from this repository, run the same command again from the Proxmox shell and choose **Update** when prompted:

```bash
bash -c "$(wget -qLO - https://raw.githubusercontent.com/C0NN0RAD0/SparkyFitness/main/install/proxmox.sh)"
```

The update script will:
1. Stop services
2. Back up your uploaded files and database backups
3. Download and deploy the latest release from this repository
4. Reinstall dependencies and rebuild the frontend
5. Restore backed-up data and restart services

> **Note:** The update command fetches the latest **GitHub Release** from `C0NN0RAD0/SparkyFitness`. Make sure a release exists in the repository before running an update.

## Script Source

The script at `install/proxmox.sh` is a modified version of the [community-scripts ProxmoxVE helper script](https://community-scripts.github.io/ProxmoxVE/scripts?id=sparkyfitness), updated to install and update from this fork (`C0NN0RAD0/SparkyFitness`) instead of the upstream project.
